# timepill-backup
胶囊日记备份程序
1. [下载python](https://www.python.org/)
2. [下载git](https://git-scm.com/downloads)
3. 打开一个命令行窗口（terminal）
4. 下载代码   
   `git clone https://github.com/gongyisheng/timepill-backup.git`
5. 进入project文件夹  
   `cd timepill-backup`
6. 安装依赖  
   `pip install -r requirement.txt`
7. 启动 **main.py** 程序  
   `python main.py`
